# ⏰ Digital Clock using Python & Tkinter

A simple GUI Digital Clock built with Python and Tkinter that displays the current time and date in real time.

## 🚀 Features
- Live time updates every second
- Displays current date
- Simple and clean GUI
- Beginner-friendly project

## 🛠️ Tech Stack
- Python
- Tkinter

## 📸 Output

-Displays a digital clock with current time and date.

📚 Learning Purpose

-This project helped me understand:
-Tkinter GUI basics
-Real-time updates using after()
-Time formatting using strftime()
